# Atlassian Intelligence

## Summary

Atlassian Intelligence is Atlassian's AI assistant embedded in Jira and other Atlassian products. It provides in-editor content generation, summaries, tone adjustments, and suggestions to accelerate user workflows.

## Key Facts

- **Core Capabilities**: In-editor content generation, tone adjustment (make professional, casual, etc.), issue summarization, definition lookups
- **Use Cases**: Generate issue descriptions from natural language, summarize long comment threads, suggest related issues, draft release notes
- **Features**: Highlights key context, identifies decisions and pending action items in issue discussions
- **Integration**: Embedded directly in Jira issue editors; no context switching needed
- **Model**: Based on LLM APIs (likely OpenAI or Anthropic) integrated into Atlassian services

## Implications

- **Competitive Parity**: AI assistance is becoming table-stakes; new platforms must offer comparable or better AI features
- **User Productivity**: AI features that save time (summarization, content generation) drive adoption and retention
- **Differentiation**: Atlassian's AI is early-stage; competitors can leapfrog with more sophisticated AI (better summarization, automated triage, semantic linking)
