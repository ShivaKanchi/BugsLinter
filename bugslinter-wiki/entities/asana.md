# Asana

## Summary

Asana is a cross-functional work management and collaboration platform emphasizing intuitive UI, visual workflows, and portfolio-level reporting. It targets marketing, operations, and product teams more than development.

## Key Facts

- **Core Features**: Intuitive UI, Work Graph model, flexible views (list/board/timeline), strong portfolio reporting, task summaries
- **Target Users**: Marketing, operations, product teams; less suitable for development workflows
- **AI Features**: Basic AI for task summaries and status update drafting; features emerging
- **Integrations**: 200+ integrations with Slack, Salesforce, and other business tools
- **Pricing**: Advanced tier ~$24.99/user/month (2025)
- **Strengths**: User-friendly interface, visual planning, portfolio/business metrics
- **Weaknesses**: Limited development workflow support, no built-in sprints, limited Git integration; performance can degrade at scale

## Implications

- **Market Differentiation**: Asana succeeds with non-technical users; development tools should focus on engineering-first UX
- **Feature Parity**: Developers accustomed to Jira often find Asana lacking for software-specific workflows
- **Growth Opportunity**: Cross-functional team growing; new tools targeting hybrid teams (dev + ops + product) could find success
